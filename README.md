
# Study Sphere

Study Sphere is a peer-learning platform powered by AI, offering interactive quizzes, coding challenges, and collaborative study tools. It helps students practice, improve, and grow together in an engaging environment.
## Deployment

To deploy this project run
#### Clone Repository
```bash
git clone https://github.com/aryanbha123/study-sphere.git
```

#### Start Frontend
```bash
cd Frontend
npm i
npm run dev
open http://localhost:5173 in your browser
```

#### Start Backend
``` bash
cd Backend
npm i 
npm run dev
Server will be running on Port = 5000
```
